# AI Document Detector (Streamlit)

This project is a Streamlit-based web application for document analysis using AI techniques. It provides an interactive interface for processing and visualizing results.

---

## 🚀 Features

- Document analysis using AI logic  
- Interactive web interface with Streamlit  
- Data visualization using Plotly  
- Environment-based configuration support  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Plotly  
- Requests  
- Python-dotenv  

---

## ▶️ Run Locally

1. Install dependencies:
pip install -r requirements.txt  

2. Run the app:
streamlit run app.py  

---

## 📂 Project Structure

app.py  
ml_model.py  
verify_helper.py  
requirements.txt  

---

## ⚠️ Environment Variables

Create a `.env` file and add:

API_KEY=your_api_key_here
